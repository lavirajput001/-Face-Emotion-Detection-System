## Face Recognition System

A Face Recognition System built using Computer Vision and Machine Learning techniques. This project detects and recognizes human faces in images or live video streams.

## Features

- Face Detection
- Face Recognition
- Real-time Webcam Support
- Easy-to-use Interface
- High Accuracy Recognition
- Support for Multiple Users

## Technologies Used

- Python
- OpenCV
- NumPy
- Face Recognition Library
- Machine Learning

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/Face-Recognition-System.git
```

2. Navigate to the project folder:

```bash
cd Face-Recognition-System
```

3. Install dependencies:

```bash
pip install -r requirements.txt
```

## Usage

### Train the Model

```bash
python train.py
```

### Run Face Recognition

```bash
python recognize.py
```

or

```bash
python main.py
```

## How It Works

1. Capture and store face images in the dataset.
2. Train the recognition model using the stored images.
3. Start the recognition system.
4. The system detects faces and matches them with the trained dataset.
5. Recognized users are displayed with their names.
